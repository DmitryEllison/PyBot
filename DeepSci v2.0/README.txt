Рефакторинг кода чат бота.
Теперь можно смотреть на код без слез.

Answers.txt - файл с ответами на сообщение.
Его нужно заполнять согласно примеру без пробелов между 
"Message", ":" и "Answer"  -   Message1:Answer1;Message2:Answer2;...

BadWords - файл с плохими словами, на которые чатбот реагирует одинаково.
Все слова пишутся через запятые. Программа не обрабатывает словосочетния, 
только слова.

Physics.txt - файл с ссылками на лекции по физике. На данный момент она не работает.
